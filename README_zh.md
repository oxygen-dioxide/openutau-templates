# OpenUTAU模板库
适用于不同UTAU引擎flag的[OpenUTAU](https://github.com/stakira/OpenUtau)工程模板。

本仓库欢迎Pull request。你可以从以下方面为本仓库进行贡献：
- 多语种本地化
- 支持更多引擎的参数
- 优化参数的命名与简称（使用更贴切的词语，以更好地，更直观地描述参数的效果）
